# Winbows11

就像 Windows11 一樣，**Winbows11** 讓您在網頁瀏覽器上體驗 Windows11 UI 和動畫！

## 注意

> [!IMPORTANT]
> 該項目**與 Microsoft 沒有任何關聯**，且**不應與 Microsoft 的作業系統**或產品混淆。 <br>
> 這不是 **Windows 365** 雲端電腦，這是 **Winbows11**。

## 正在開發中

Windows11目前正在開發中，因此某些功能尚不可用，但不用擔心，很快就會被完成！
